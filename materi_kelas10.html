Materi Pemrograman Dasar Kelas X

Pemrograman Web • 10 menit baca • Oleh Admin • 31 Jul 2025

"Kode adalah puisi teknologi, menari di antara baris untuk menciptakan dunia baru."

1. Python Dasar

Python adalah bahasa pemrograman yang sederhana namun kuat, ideal untuk pemula. Berikut adalah konsep dasar yang perlu dikuasai.

  • Variabel dan Input/Output

    Mengambil input dari pengguna dan menampilkannya.

    nama = input("Masukkan nama: ")
print("Halo", nama)
  • Percabangan

    Menggunakan if untuk pengambilan keputusan.

    x = 10
if x > 5:
    print("Lebih besar dari 5")
  • Perulangan

    Menggunakan for untuk iterasi berulang.

    for i in range(5):
    print("Iterasi ke-", i)

2. HTML

HTML adalah tulang punggung halaman web, digunakan untuk membuat struktur konten.

  • Struktur Dasar

    Dasar setiap dokumen HTML.

    <!DOCTYPE html>
<html>
  <head>
    <title>Judul</title>
  </head>
  <body>
    Halo Dunia
  </body>
</html>
  • Formulir

    Membuat formulir untuk input pengguna.

    <form action="proses.php" method="post">
  <input type="text" name="nama" />
  <button type="submit">Kirim</button>
</form>

3. CSS

CSS digunakan untuk menata tampilan halaman web agar lebih menarik.

  • Box Styling

    Mengatur padding, margin, dan border elemen.

    div {
  padding: 20px;
  margin: 10px;
  border: 1px solid black;
}
  • Flexbox

    Mengatur tata letak elemen secara fleksibel.

    .container {
  display: flex;
  justify-content: space-between;
}

4. PHP Dasar

PHP adalah bahasa server-side untuk membangun web dinamis dan terhubung dengan database.

  • Menampilkan Teks

    Menampilkan output sederhana di halaman web.

    <?php
echo "Halo Dunia";
?>
  • Koneksi Database

    Menghubungkan PHP dengan database MySQL.

    <?php
$conn = mysqli_connect("localhost", "root", "", "sekolah");
if (!$conn) {
    die("Gagal terhubung: " . mysqli_connect_error());
}
?>

5. Java Dasar

Java adalah bahasa pemrograman serbaguna, sering digunakan untuk aplikasi skala besar.

  • Program Dasar

    Struktur dasar program Java.

    public class Halo {
  public static void main(String[] args) {
    System.out.println("Halo Dunia");
  }
}
  • OOP Sederhana

    Konsep dasar pemrograman berorientasi objek.

    class Mobil {
  String merk;
  Mobil(String m) {
    merk = m;
  }
  void jalan() {
    System.out.println(merk + " berjalan");
  }
}

6. API Laravel (Bonus)

Pengenalan singkat membuat API sederhana dengan Laravel untuk pengelolaan data siswa.

  • Routing API

    Menentukan rute API di Laravel.

    // routes/api.php
use App\Http\Controllers\SiswaController;
Route::get('/siswa', [SiswaController::class, 'index']);
  • Controller

    Menangani logika untuk endpoint API.

    namespace App\Http\Controllers;
use App\Models\Siswa;
use Illuminate\Http\Request;

class SiswaController extends Controller {
  public function index() {
    return response()->json(Siswa::all(), 200);
  }
}
  • Model dan Migration

    Membuat model dan migration untuk tabel siswa.

    // terminal
php artisan make:model Siswa -m
    // database/migrations/xxxx_create_siswa_table.php
use Illuminate\Database\Migrations\Migration;
use Illuminate\Database\Schema\Blueprint;
use Illuminate\Support\Facades\Schema;

class CreateSiswaTable extends Migration {
  public function up() {
    Schema::create('siswa', function (Blueprint $table) {
      $table->id();
      $table->string('nama');
      $table->timestamps();
    });
  }
  public function down() {
    Schema::dropIfExists('siswa');
  }
}
