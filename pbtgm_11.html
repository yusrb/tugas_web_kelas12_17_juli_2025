Pemrograman Berbasis Teks dan Grafis Multimedia

PBTGM • 8 menit baca • Oleh Admin • 29 Jul 2025

"Pembelajaran desktop C# bersanding SQL untuk membangun aplikasi nyata."

Pendahuluan

PBTGM kelas XI memperkenalkan dasar-dasar pemrograman menggunakan C# untuk aplikasi desktop. Dengan antarmuka Windows Forms dan penyimpanan berbasis SQL Server, siswa diajak menciptakan aplikasi yang nyata dan fungsional.

Apa itu C# Desktop?

C# (dibaca: C-sharp) adalah bahasa pemrograman dari Microsoft. Dikombinasikan dengan Windows Forms, C# memungkinkan pembuatan aplikasi GUI yang profesional dan interaktif.

Contoh Program Hello World

using System;
using System.Windows.Forms;

public class Hello : Form {
  public Hello() {
    Label label = new Label();
    label.Text = "Halo Dunia!";
    label.Dock = DockStyle.Fill;
    label.TextAlign = System.Drawing.ContentAlignment.MiddleCenter;
    this.Controls.Add(label);
  }

  [STAThread]
  public static void Main() {
    Application.Run(new Hello());
  }
}

Integrasi SQL Server dan CRUD

Untuk menyimpan data secara efisien, aplikasi C# dihubungkan dengan SQL Server. Proses ini mencakup operasi CRUD: Create, Read, Update, dan Delete.

Langkah-Langkah

  1. Pasang SQL Server dan SQL Server Management Studio (SSMS).
  2. Buat database bernama db_sekolah.
  3. Buat tabel siswa: 
    CREATE TABLE siswa (
  id INT PRIMARY KEY IDENTITY(1,1),
  nama NVARCHAR(100),
  nis VARCHAR(20),
  kelas VARCHAR(10)
);
  4. Buka Visual Studio, buat proyek Windows Forms App (.NET Framework).

Contoh Koneksi dan Baca Data

using System;
using System.Data.SqlClient;
using System.Windows.Forms;

namespace MyApp {
  public partial class Form1 : Form {
    public Form1() {
      InitializeComponent();
      LoadData();
    }

    private void LoadData() {
      string connStr = "Server=localhost;Database=db_sekolah;Trusted_Connection=True;";
      using (SqlConnection conn = new SqlConnection(connStr)) {
        try {
          conn.Open();
          MessageBox.Show("Terhubung ke database!");
          string query = "SELECT * FROM siswa";
          using (SqlCommand cmd = new SqlCommand(query, conn)) {
            using (SqlDataReader reader = cmd.ExecuteReader()) {
              while (reader.Read()) {
                Console.WriteLine(reader["nama"]);
              }
            }
          }
        } catch (Exception ex) {
          MessageBox.Show("Gagal: " + ex.Message);
        }
      }
    }
  }
}

Contoh Fungsi CRUD

using System.Data.SqlClient;

// CREATE
string connStr = "Server=localhost;Database=db_sekolah;Trusted_Connection=True;";
using (SqlConnection conn = new SqlConnection(connStr)) {
  conn.Open();
  string insertQuery = "INSERT INTO siswa (nama, nis, kelas) VALUES (@nama, @nis, @kelas)";
  using (SqlCommand insertCmd = new SqlCommand(insertQuery, conn)) {
    insertCmd.Parameters.AddWithValue("@nama", "Ayu");
    insertCmd.Parameters.AddWithValue("@nis", "112233");
    insertCmd.Parameters.AddWithValue("@kelas", "XI-RPL");
    insertCmd.ExecuteNonQuery();
  }

  // READ
  string readQuery = "SELECT * FROM siswa";
  using (SqlCommand readCmd = new SqlCommand(readQuery, conn)) {
    using (SqlDataReader r = readCmd.ExecuteReader()) {
      while (r.Read()) {
        Console.WriteLine($"{r["nama"]} - {r["kelas"]}");
      }
    }
  }

  // UPDATE
  string updateQuery = "UPDATE siswa SET kelas=@kelas WHERE nis=@nis";
  using (SqlCommand updateCmd = new SqlCommand(updateQuery, conn)) {
    updateCmd.Parameters.AddWithValue("@kelas", "XI-MM");
    updateCmd.Parameters.AddWithValue("@nis", "112233");
    updateCmd.ExecuteNonQuery();
  }

  // DELETE
  string deleteQuery = "DELETE FROM siswa WHERE nis=@nis";
  using (SqlCommand deleteCmd = new SqlCommand(deleteQuery, conn)) {
    deleteCmd.Parameters.AddWithValue("@nis", "112233");
    deleteCmd.ExecuteNonQuery();
  }
}
