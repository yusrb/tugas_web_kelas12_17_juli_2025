Pemrograman Perangkat Bergerak (PPB)

PPB • 8 menit baca • Oleh Admin • 28 Jul 2025

"Dart menari, aplikasi menjelma — satu basis, dua dunia (Android & iOS)."

1. Pendahuluan

Flutter, framework modern dari Google, memadukan keindahan antarmuka pengguna dengan efisiensi logika menggunakan bahasa Dart. Satu kode dapat berjalan di Android dan iOS, menghemat waktu dan usaha.

2. Dasar Bahasa Dart

Dart adalah bahasa yang cepat dan mendukung pemrograman berorientasi objek. Berikut adalah konsep dasarnya:

  • Variabel: var nama = "Tuan";
  • Fungsi: void sapa() { print("Halo"); }
  • Kondisi: Menggunakan if, else, switch.
  • Perulangan: Menggunakan for, while, do-while.
  • Kelas dan Objek: Mendukung OOP (Class, Constructor, Method, Inheritance).
void main() {
  var nama = 'Tuan';
  print('Selamat datang, $nama');
}

3. Widget Dasar

Flutter membangun antarmuka dari Widget, komponen kecil yang membentuk seluruh tampilan. 

import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        body: Center(child: Text('Halo, Flutter!')),
      ),
    );
  }
}

4. Tata Letak (Layout)

Tata letak diatur menggunakan widget seperti Column, Row, dan Container untuk menyusun elemen secara fleksibel. 

Column(
  children: [
    Text('Judul', style: TextStyle(fontSize: 20)),
    SizedBox(height: 10),
    Row(
      children: [
        Icon(Icons.star, color: Colors.amber),
        Text('5.0'),
      ],
    ),
  ],
)

5. Stateful Widget

StatefulWidget digunakan untuk widget dengan data yang dapat berubah, seperti penghitung interaktif. 

import 'package:flutter/material.dart';

void main() => runApp(MaterialApp(home: CounterApp()));

class CounterApp extends StatefulWidget {
  @override
  _CounterAppState createState() => _CounterAppState();
}

class _CounterAppState extends State {
  int count = 0;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text('Angka: $count'),
            ElevatedButton(
              onPressed: () => setState(() => count++),
              child: Text('Tambah'),
            )
          ],
        ),
      ),
    );
  }
}

6. Navigasi dan Routing

Navigasi antar halaman dilakukan dengan Navigator dan konfigurasi rute menggunakan MaterialApp. 

Navigator.push(
  context,
  MaterialPageRoute(builder: (context) => HalamanKedua()),
);

MaterialApp(
  initialRoute: '/',
  routes: {
    '/': (context) => Beranda(),
    '/kedua': (context) => HalamanKedua(),
  },
);

7. Packages Tambahan

Tambahkan fungsionalitas seperti HTTP, state management, atau font dengan mengedit pubspec.yaml. 

dependencies:
  flutter:
    sdk: flutter
  http: ^0.13.5
  provider: ^6.0.5
  google_fonts: ^6.1.0

8. Penutup

Dari Dart hingga Flutter, dari statis ke dinamis, dari satu layar ke banyak halaman — inilah seni PPB. Tuan dan para siswa diajak menjelajahi dunia pemrograman modern yang indah dan tangkas.
