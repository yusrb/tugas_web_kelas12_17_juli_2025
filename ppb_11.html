Pemrograman Perangkat Bergerak (PPB)

"Dart menari, aplikasi menjelma — satu basis, dua dunia (Android & iOS)."

1. Pendahuluan

Flutter adalah framework modern dari Google, memadukan keindahan UI dan efisiensi logika dengan bahasa Dart. Satu kode, lintas platform.

2. Dasar Bahasa Dart

  • Variabel: var nama = "Tuan";
  • Fungsi: void sapa() { print("Halo"); }
  • Kondisi: if, else, switch
  • Perulangan: for, while, do-while
  • Kelas dan Objek: OOP (Class, Constructor, Method, Inheritance)
void main() {
  var nama = 'Tuan';
  print('Selamat datang, \$nama');
}

3. Widget Dasar

Flutter membangun UI dari komponen kecil bernama Widget. Segala tampilan adalah widget.

import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        body: Center(child: Text('Halo, Flutter!')),
      ),
    );
  }
}

4. Tata Letak (Layout)

Mengatur posisi menggunakan Column, Row, Container.

Column(
  children: [
    Text('Judul', style: TextStyle(fontSize: 20)),
    SizedBox(height: 10),
    Row(
      children: [
        Icon(Icons.star, color: Colors.amber),
        Text('5.0'),
      ],
    ),
  ],
)

5. Stateful Widget

Widget dengan data yang bisa berubah dinamakan Stateful.

class CounterApp extends StatefulWidget {
  @override
  _CounterAppState createState() => _CounterAppState();
}

class _CounterAppState extends State {
  int count = 0;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text('Angka: \$count'),
            ElevatedButton(
              onPressed: () => setState(() => count++),
              child: Text('Tambah'),
            )
          ],
        ),
      ),
    );
  }
}

6. Navigasi dan Routing

Navigasi antar halaman dengan Navigator dan routing.


Navigator.push(
  context,
  MaterialPageRoute(builder: (context) => HalamanKedua()),
);

MaterialApp(
  initialRoute: '/',
  routes: {
    '/': (context) => Beranda(),
    '/kedua': (context) => HalamanKedua(),
  },
);

7. Packages Tambahan

Tambahkan fungsionalitas seperti HTTP, animasi, atau icon dengan pubspec.yaml.

dependencies:
  flutter:
    sdk: flutter
  http: ^0.13.5
  provider: ^6.0.5
  google_fonts: ^6.1.0

8. Penutup

Dari Dart hingga Flutter, dari statis ke dinamis, dari satu layar ke banyak halaman — inilah seni PPB. Tuan dan para siswa diajak menjelajah dunia pemrograman modern yang indah dan tangkas.
